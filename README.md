# 🍰 Doce Momento

Aplicação web para venda de bolos artesanais com carrinho de compras, formulário de entrega e envio automático de pedidos via WhatsApp.

---

# Sistema online
 virtual-cakes.vercel.app
---

## 🚀 Funcionalidades

- Carrinho de compras dinâmico  
- Controle de quantidade de produtos  
- Remoção completa de itens  
- Cálculo automático do total  
- Formulário de endereço em modal  
- Validação de campos obrigatórios  
- Sistema de alertas centralizado  
- Geração automática de pedido no WhatsApp  

---

## 🧠 Como funciona

1. O usuário escolhe os produtos  
2. Adiciona ao carrinho  
3. Clica em **Finalizar Pedido**  
4. Preenche o endereço de entrega  
5. O sistema gera a mensagem do pedido  
6. O usuário é redirecionado para o WhatsApp com tudo pronto

## 📲 Integração com WhatsApp

Os pedidos são enviados automaticamente através do link:

https://wa.me/NUMERO  

Com mensagem contendo:
- Lista de produtos  
- Quantidade  
- Total  
- Endereço de entrega
